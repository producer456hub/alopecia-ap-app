# Homeostasis & Alopecia — A&P Interactive

An interactive web app for anatomy & physiology students exploring homeostasis concepts through alopecia (hair loss).

## Topics Covered
- Set point & normal range
- Stimulus, Receptor, Control Center, Effector
- Negative feedback loops (HPG axis / androgen regulation)
- Positive feedback loops (AGA miniaturization spiral, AA autoimmune loop)
- Three types of alopecia: androgenetic, alopecia areata, telogen effluvium
- Interactive drug simulator (finasteride, dutasteride, minoxidil, spironolactone, JAK inhibitors, PRP)
- Two quizzes: concept-focused and alopecia content

## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `alopecia-ap-app`)
2. Upload `index.html` to the root of the repo
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch` → `main` → `/ (root)`
5. Click **Save** — your site will be live at `https://yourusername.github.io/alopecia-ap-app/`

## Local Use
Just open `index.html` in any modern browser. No build step, no dependencies.
